## Quick Start

Get started with React Pro Sidebar in just a few lines of code. Install the package
and start building your navigation interface.

```bash
npm install react-pro-sidebar
```

```jsx
import { Sidebar, Menu, MenuItem } from 'react-pro-sidebar';

function App() {
  return (
    <Sidebar>
      <Menu>
        <MenuItem> Dashboard </MenuItem>
        <MenuItem> Calendar </MenuItem>
      </Menu>
    </Sidebar>
  );
}
```
